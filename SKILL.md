# X (Twitter) Stealth Scraper
Description: A tool to read posts from x.com and twitter.com, bypassing login walls.

## Tools

### read_x_post
- Description: Use this tool WHENEVER the user provides a URL containing "x.com" or "twitter.com". It retrieves the full text, author, and timestamp of the post.
- Parameters:
  - url: The full URL of the tweet/post (string, required)
